        </font>
        </h2>
    <p align="center">
        <em>
        <font color="#666666">
            2019.04.21&nbsp;&nbsp;&nbsp;&nbsp;地点：塔克拉玛干沙漠
        </font>
        </em>
        </p>
    <hr color="#CCC"/>
    <p>
        &nbsp;&nbsp;&nbsp;&nbsp;在这浩瀚的大漠我看到了姿态各异、生命力顽强的胡杨。没有肥沃的土壤、没有充足的水分、没有适宜的环境，面对着寸草不生的塔克拉玛干大沙漠，却有胡杨毅然傲立戈壁，呈现出生命醉人的绿。
    </p>
